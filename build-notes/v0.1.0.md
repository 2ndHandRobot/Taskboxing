# Taskboxing — v0.1.0

Initial build. Chrome extension for task management with Google Calendar integration.

---

## Foundation
- UI store with list filter/sort and editor state
- Layout shell with SideRail navigation, mount sequence, and view routing
- SVG icon system

## Task Management
- `TaskGroup` collapsible component with `TaskItem` two-line display, accent border, and completion toggle
- `DependencyPicker` with live task search and removable chips
- `SubtaskList` with drag-to-reorder via dnd-kit and inline add
- `TimeConstraintSelector` with all 9 constraint variants and conditional inputs
- Full `TaskEditorModal` with all form fields
- Inline `TaskItemEditor` with expand-to-modal button
- `ListView` with grouping, filters, and task editor integration

## Calendar Views
- `CalendarView` with month navigation and `MonthGrid`
- `DayCell` with event pills, task chips, and `EventPopover`
- `WeekView` and `DayView` implementations
- Calendar mode tabs, task creation list/due-date, icon-opens-side-panel fixes
- Due date pre-fill, task list selector fixes

## Google Calendar Integration
- `defaultSchedulingCalendarId` setting and `createCalendar` API method
- Event completion state and `createTaskEvent` in calendar store
- `useCompleteTask` and `useCompleteEvent` hooks
- Mark done/incomplete and linked task info in `EventPopover`
- Hover checkbox and completed visual state on calendar event chips
- Settings panel with scheduling calendar selector

## Manual Scheduling
- `DayPreview` and `ScheduleForm` components
- `ScheduleForm` integrated into task editor
- Cross-calendar reschedule logic (inverted order, zero-duration guard)
- Unschedule always clears task metadata even if GCal delete fails
- `EventPopover` anchored to clicked event across all calendar views
- Single-popover-at-a-time enforcement
